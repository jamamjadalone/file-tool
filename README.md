# File Extension Changer Tool

A free, browser-based tool to change file extensions to JPG, PNG, PDF, ZIP, TXT, or custom formats.

## Features
- Change multiple file extensions at once
- Supports JPG, PNG, PDF, ZIP, TXT formats
- Custom extension support
- No installation required
- Works locally in your browser
- Completely free and safe

## How to Use
1. Visit the live demo link
2. Click "Select Folder" to choose files
3. Select target extension (JPG, PNG, etc.)
4. Click "Change Extensions"
5. Download your renamed files
